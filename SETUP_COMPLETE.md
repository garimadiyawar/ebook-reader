## 🛠️ How It Works

### File Upload Flow
1. User selects MD/EPUB/MOBI file
2. bookParser.ts extracts text and chapters
3. Book stored in browser's IndexedDB (local storage)
4. Appears in library immediately

### Reading & Listening
1. User selects a book
2. Reader displays chapter with controls
3. User clicks "Play"
4. textToSpeech.ts uses Web Speech API
5. Browser speaks the text naturally
6. Position saved automatically

### Offline Support
1. Service worker caches app on first load
2. IndexedDB stores all books locally
3. Works perfectly without internet
4. Perfect for the gym! 💪

## 🐛 Troubleshooting

**Issue: Books don't persist**
- Make sure PWA is in standalone mode
- Try adding to home screen again

**Issue: TTS not working**
- Check volume isn't muted
- Try reloading the app
- Works best in Safari on iOS

**Issue: EPUB not parsing**
- File might be corrupted
- Try converting to MD or MOBI
- Max file size ~50MB

**Issue: Can't hear voice**
- Increase volume control in app
- Check device isn't muted
- Try different pitch/speed
